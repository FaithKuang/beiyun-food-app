# 备孕饭点

一个给少数人自用的 iPhone PWA，用来做备孕/孕期阶段饮食提醒、每日打卡和每周采购清单。

## 当前功能

- 选择使用者、性别和阶段。
- 阶段包括备孕前三个月、排卵/同房窗口、孕早期、孕中期、孕晚期。
- 按阶段展示今日重点营养和可勾选食物。
- 本地保存每日打卡、阶段、忌口、提醒和使用时长，不需要账号和同步。
- 支持显式保存按钮，保存后会显示保存时间。
- 支持最近一周饮食记录回顾和每周分析报告。
- 根据当前阶段生成一周山姆采购清单和一周饮食建议。
- 支持简单忌口过滤。
- 支持浏览器通知提醒。当前提醒依赖网页或主屏幕 App 运行，长期后台推送可作为下一版升级。
- 使用绿色主视觉、卡通食物贴纸和夫妻用餐图标。

## 本地运行

```powershell
python -m http.server 4173 --bind 127.0.0.1
```

然后打开：

```text
http://127.0.0.1:4173/index.html
```

## iPhone 使用方式

1. 将这个静态站点部署到 HTTPS 地址，例如 Vercel、Netlify、Cloudflare Pages 或自己的服务器。
2. 把部署后的链接发给你老婆。
3. 两台 iPhone 都用 Safari 打开链接。
4. 点分享按钮，选择“添加到主屏幕”。
5. 从主屏幕打开后，进入“设置”开启提醒。

本地的 `http://127.0.0.1:4173/index.html` 只能在当前电脑上预览，不能直接发给别人使用。

## 推荐交付方式

最简单的交付是 PWA：不需要 App Store，不需要审核，也不需要安装证书。用户看到的是一个桌面图标，打开后像普通 App 一样全屏运行。

如果一定要做成原生 iOS App 文件，也可以走 TestFlight 或 Ad Hoc，但通常需要 Apple Developer 账号、设备登记或测试审核，不适合两三个人快速自用。

## 内容边界

这里的内容是饮食提醒和记录，不替代医生建议。备孕、孕期、有基础疾病或正在服药时，以医生或营养师建议为准。
